База данных online_shop создана для обеспечения работы онлайн магазина.

Описание таблиц:
1) city_list - список имён городов.
     id - номер города
     name - имя города 

2) product_category - список категорий товаров онлайн магазина.
     id - номер категории
     name - имя категории товаров

3) warehouses - список складов онлайн магазина.
     id serial - номер склада
     city_id - номер города из таблицы city_list
     phone_number - номер телефона

4) vendor_list - список поставщиков онлайн магазина.
     id - номер поставщика
     name - имя поставщика
     phone_number - номер телефона поставщика
     email - электронный адрес поставщика
     address - адрес поставщика
     city_id - номер города из таблицы city_list

5) products - таблица товара. 
     id - номер товара 
     name - имя товара 
     image_url - url фото товара 
     price - цена товара 
     articulus - артикуль товара
     product_category_id - номер категории товара 
     product_type - тип товара (товар, услуга)
     product_count - текущий остаток на складе
     city_id - номер города нахождения товара из таблицы city_list
     vendor_id - номер поставщика из таблицы vendor_list

6) current_amount - таблица текущего складского остатка.	
     product_id - номер продукта из таблицы products 
     warehouses_id - номер склада из таблицы warehouses 
     amount - количество товара 

7) users - таблица пользователей.
     id - номер пользователя
     name - имя пользователя
     phone_number - номер телефона
     email - адрес электронной почты
     address - адрес пользователя
     city_id - номер города нахождения товара из таблицы city_list
     user_type - тип пользователя(админ,покупатель)

8) operation_types - типы операций с товарами.
     id - номер типа операции
     name - название операции

9) operation_list - список операций с товарами 
     id - номер операции
     product_id - номер продукта из таблицы products
     operation_type - номер типа операции из таблицы operation_types 
     created_at - дата операции

10) position_list - таблица должностей сотрудников онлайн магазина.
     id - номер должности сотрудника	
     position_name - имя позиции сотрудника

11) staff_list - таблица работников онлайн магазина.
     id - номер сотрудника
     first_name - имя сотрудника
     lastname - фамилия сотрудника
     position - номер должности сотрудника из таблицы position_list 
     status - статус сотрудника(работает, уволен, в отпуске, болеет)
     warehouses_id - номер скалада где работает сотрадник из таблицы warehouses

12) delivery_types - таблица видов доставок.
     id - номер доставки
     delivery_name - название доставки
     delivery_terms - условия доставки
     company_name - имя компании осуществляющую доставку
     price - стоимость доставки

13) orders - таблица заказов.
     id - номер заказа
     user_id - номер заказавшего пользователя
     city_id - номер города заказа из таблицы city_list
     delivery_type - номер типа доставки из таблицы delivery_types

14) specifications - спецификация заказа (один заказ - несколько спецификаций).
	id - номер спецификации
	product_id - id товара 
	order_id - номер заказа
	`count` - количество товара
	`price` - цена товара 
	`amount` - итоговая сумма
